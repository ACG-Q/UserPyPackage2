# UserPyPackage

一个简易的基于 Github Issue 为主的Python自动构建系统

## 功能

1. 多平台 pyinstaller打包
   1. 单文件打包
   2. 文件夹压缩上传

### 示例

1. 多平台打包演示

   issue: [\[打包请求\] 你的项目名称](https://github.com/ACG-Q/UserPyPackage2/issues/5)

   release: [打包结果](https://github.com/ACG-Q/UserPyPackage2/releases/tag/v20241121072422935)
